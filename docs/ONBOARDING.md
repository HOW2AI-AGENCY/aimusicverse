## Шаг 2: Настройте свое рабочее окружение

### 2.1. Системные требования
- `Node.js >= 18.0.0`
- `npm >= 9.0.0`

### 2.2. Установка
```bash
# 1. Клонируйте репозиторий
git clone https://github.com/HOW2AI-AGENCY/aimusicverse.git
cd aimusicverse

# 2. Установите зависимости
npm install
```

### 2.3. Переменные окружения
Для локальной разработки вам понадобится файл `.env`. Вы можете создать его, скопировав пример:
```bash
cp .env.example .env
```
Затем откройте `.env` и заполните его необходимыми значениями:
- `VITE_SUPABASE_URL`: URL вашего проекта в Supabase.
- `VITE_SUPABASE_ANON_KEY`: Публичный ключ вашего проекта в Supabase.

### 2.4. Запуск в режиме разработки (Mock Mode)
Наш проект можно запустить в браузере без необходимости отладки через Telegram. В этом режиме SDK Telegram эмулируется, что позволяет разрабатывать и тестировать интерфейс.

```bash
# Запустить сервер для разработки
npm run dev
```
После запуска приложение будет доступно по адресу `http://localhost:8080`.
