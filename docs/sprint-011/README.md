## üéØ Documentation Standards

### User Guides
- **Audience**: End users (non-technical)
- **Language**: Russian (primary) with English fallback
- **Format**: Step-by-step with screenshots
- **Length**: 500-1500 words per guide
- **Structure**:
  - Introduction (What is this feature?)
  - Quick Start (3-5 steps to get started)
  - Detailed Guide (All features explained)
  - FAQ (Common questions)
  - Tips & Best Practices

### API Reference
- **Audience**: Developers
- **Language**: English
- **Format**: Reference style with code examples
- **Length**: 1000-3000 words per reference
- **Structure**:
  - Overview
  - Authentication & Authorization
  - Endpoints/Functions
  - Request/Response Examples
  - Error Handling
  - Rate Limits
  - Code Samples (TypeScript, cURL)
