# Color Contract

**Feature**: 032-professional-ui

## CSS Variables

:root {
  --color-primary: #6366f1;
  --color-secondary: #8b5cf6;
  --color-success: #10b981;
  --color-warning: #f59e0b;
  --color-error: #ef4444;
}

.dark {
  --color-primary: #818cf8;
  --color-secondary: #a78bfa;
}

## Tailwind Classes

- `bg-primary` - Primary action backgrounds
- `text-primary` - Primary action text/links
- `bg-gradient-primary` - Gradient backgrounds
- `from-primary to-secondary` - Gradient directions

## Usage

Use semantic color names, not direct hex values.
