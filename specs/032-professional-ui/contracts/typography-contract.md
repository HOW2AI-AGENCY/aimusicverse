# Typography Contract

**Feature**: 032-professional-ui

## Class Names

- `.text-h1` - 28px, bold, 1.2 lh - Page titles
- `.text-h2` - 24px, semibold, 1.3 lh - Section headers
- `.text-h3` - 20px, semibold, 1.3 lh - Card titles
- `.text-h4` - 18px, medium, 1.4 lh - List titles
- `.text-body` - 16px, regular, 1.5 lh - Primary content
- `.text-body-sm` - 14px, regular, 1.5 lh - Standard text
- `.text-caption` - 12px, regular, 1.4 lh - Metadata

## Usage

Apply to typography elements via Tailwind classes or extend in theme.
