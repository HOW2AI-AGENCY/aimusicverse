### Usage

Colors are applied using Tailwind CSS utility classes. For example, to set the background color to the primary color, you would use the class `bg-primary`.
The system is built on top of CSS variables, which allows for easy theming. You can find the variable definitions in `:root` and `.dark` selectors in `src/index.css`.
